# 📦 LocalStorage & SessionStorage Project

##  Descrição

Este projeto demonstra o uso de **LocalStorage** e **SessionStorage** no navegador para armazenar dados do utilizador.

O sistema permite:

* Guardar o nome do utilizador
* Mostrar mensagem de boas-vindas
* Simular login e logout simples

---

## Tecnologias usadas

* HTML5
* JavaScript (Vanilla)
* Web Storage API

---

##  Funcionalidades

* ✅ Armazenamento com LocalStorage
* ✅ Persistência de dados após recarregar a página
* ✅ Logout (remoção de dados)
* ✅ Interface simples e funcional
